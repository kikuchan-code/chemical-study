脂肪族炭化水素のいろは

呼び名（日本語） 呼び名（英語） 一般式 説明
アルカン alkane CnH2n+2 (n ≥ 1) 単結合のみをもつ 飽和炭化水素。 メタン系炭化水素とも。
n ≥ 5の場合、 「(接頭語-a)+ ane」で命名される。
アルケン alkene CnH2n (n ≥ 2) 二重結合を1つもつ 不飽和炭化水素。 エチレン系炭化水素とも。
n ≥ 5の場合、 「(接頭語-a)+ ene」で命名される。
アルキン alkyne CnH2n-2 (n ≥ 2) 三重結合を1つもつ 不飽和炭化水素。 アセチレン系炭化水素とも。
n ≥ 5の場合、 「(接頭語-a)+ yne」で命名される。
アルキル基 alkyl基 CnH2n+1 (n ≥ 1) アルカンから 1つの水素原子 を取り除いた構造。
n ≥ 5の場合、 「(接頭語-a)+ yl基(group)」 で命名される。
入力後「正誤判定」で採点できます。
